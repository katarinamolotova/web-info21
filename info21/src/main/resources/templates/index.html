Info21 v2.0 Web

Web-интерфейса для проекта SQL2 на языке Java.

Интерфейс позволяет:

  • Делать CRUD операции над всеми таблицами;
  • Вызывать заготовленные функции и процедуры;
  • Делать импорт во все таблицы базы данных;
  • Писать самостоятельно запрос в базу данных;
  • Делать экспорт всех таблиц и результатов функций/процедур.
